CAPÍTULO 3: UV, Animação e o Salto Pro

Wikibooks (comunidade)

Para texturas reais e movimento com alma: desdobrar o modelo e animar por keyframes com curvas.

UV Unwrap

Para colar uma foto numa superfície sem distorção, o modelo é 'desdobrado' como uma caixa de papelão: marcar seams → Unwrap → ajustar as ilhas no UV Editor. É o passaporte para texturas reais.

Sem UV bom: imagens esticam e repetem feio.

Keyframe + Graph Editor

I grava posição/rotação/escala num momento; o Blender interpola o resto. O Graph Editor (FCurves) ajusta a velocidade de entrada/saída — é o que troca o 'movimento robótico' linear por movimento orgânico.

Premium: curva suave (ease) > interpolação linear pura.

O salto para Pro

O intermediário vira avançado com Node Editor (materiais/composição modulares), Cycles (luz física, GI grátis) e rigging (armature + weight paint). É a porta do pipeline profissional.

Trilha: nodes e rigging são o degrau Noob→Pro de verdade.

Lições-Chave do Capítulo 3

  • UV unwrap (seams→unwrap→ilhas) é pré-requisito de texturas reais.
  • Keyframe + Graph Editor dão movimento orgânico, não robótico.
  • Nodes, Cycles e rigging marcam o salto para o nível profissional.
