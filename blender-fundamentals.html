Blender Foundation

O curso gratuito de fundamentos da própria Blender Foundation: a forma 'certa' de aprender, dos atalhos essenciais ao workflow de produção (modelar → texturizar → iluminar → renderizar → compor). Cheat-sheet pt-BR derivado do material oficial sob CC-BY 4.0.

O workflow de produção

A ordem oficial: Modelagem → Texturização → Shading → Rigging → Animação → Iluminação → Render → Compositing. Cada Workspace do Blender corresponde a uma etapa — siga a esteira.

Modelo mental: produção é linha de montagem; pule etapa e retrabalha.

F3 é a rota de fuga

Não decore menus: aperte F3 e busque o comando pelo nome ('bevel', 'shade smooth'). É o atalho que destrava qualquer função quando você sabe o que quer, mas não onde fica.

Produtividade: F3 > caçar em submenu.

Cycles × EEVEE

Cycles = path tracer físico (lento, foto-realista). EEVEE = tempo real (rápido, ótimo p/ preview e estilizado). O curso ensina os dois — escolha pela necessidade.

Regra: EEVEE p/ iterar/preview; Cycles p/ a entrega final.

Nunca renderize direto p/ .mp4

A prática oficial: renderize a animação para uma SEQUÊNCIA de imagens (PNG/EXR), nunca direto para .mp4. Um crash no meio invalida o vídeo inteiro; a sequência preserva cada frame.

Depois: monte o filme no Video Sequencer (com áudio).
